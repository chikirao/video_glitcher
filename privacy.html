Privacy Policy

Glitchy is a static client-side media tool. Your video, photo, and audio files are processed locally in your browser for preview and export.

This site does not state that your media files are uploaded to a server operated by the project owner as part of the normal app flow.

Your browser may store basic interface preferences locally, such as theme, language, or dismissed notices, so the interface can stay consistent between visits.

If the site is served through GitHub Pages, GitHub may process standard technical request data as the hosting provider, such as IP address, user agent, and request logs.

If you do not want local browser storage for preferences, you can clear this site’s data in your browser settings.
